# Dialog 오버레이 배경 투명도 개선

*작성일: 2025-11-03*

*작성자: Designer Agent*

*이슈 유형: UI 개선*

*우선순위: Medium*

## 📋 문제 상황

### 발견된 문제점
- Dialog 오버레이 배경이 너무 진함 (`bg-black/80`)
- 뒤 배경이 거의 보이지 않아 모달 느낌이 과도함
- 사용자가 현재 페이지 컨텍스트를 잃을 수 있음

## 🔍 원인 분석

### 1. 과도한 불투명도
- `bg-black/80` (80% 불투명도)는 너무 진한 배경
- 일반적인 모달 오버레이는 50-60% 정도가 적절

### 2. 시각적 무게감
- 진한 배경으로 인해 모달이 너무 강조됨
- 뒤 배경과의 연결성 부족

## ✅ 해결 방법

### 적용된 변경사항
```tsx
// 변경 전
bg-black/80

// 변경 후
bg-black/50 backdrop-blur-sm
```

### 개선 효과
- **투명도 조정**: 80% → 50%로 변경하여 적절한 배경 가시성 확보
- **블러 효과 추가**: `backdrop-blur-sm`으로 세련된 모달 효과
- **컨텍스트 유지**: 사용자가 현재 페이지를 인지하면서도 모달에 집중

## 📊 디자인 원칙

### 모달 오버레이 가이드라인
- **투명도**: 40-60% 범위 권장
- **블러 효과**: 배경과 모달 구분을 위해 선택적 사용
- **색상**: 검은색 또는 어두운 회색 사용

### 사용자 경험
- 뒤 배경이 적절히 보여 컨텍스트 유지
- 모달 콘텐츠에 집중하면서도 페이지 이탈감 최소화
- 세련된 블러 효과로 프리미엄 느낌

## 🔄 예방 방법

### 1. 디자인 시스템 표준화
- 모달 오버레이 투명도 표준: `bg-black/50`
- 블러 효과 표준: `backdrop-blur-sm`

### 2. 일관성 유지
- 모든 Dialog/Modal 컴포넌트에 동일한 스타일 적용
- Drawer, Sheet 등 유사 컴포넌트도 동일한 원칙 적용

### 3. 사용자 테스트
- 다양한 배경에서 가독성 테스트
- 접근성 확인 (색상 대비비)
