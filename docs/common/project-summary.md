# 아파트 커뮤니티 프로젝트 목적

## 개요

- 아파트 입주민들의 생활에 필요한 전반적인 것들을 디지털화
- 기존 종이 고지서 및 종이 공지사항을 전자화 함으로써 환경 친화적인 아파트 플랫폼 추구
- 관리사무소 -> 입주민에 필요한 프로세스를 통합한 플랫폼을 만듦으로써 관리 효율성 상승
- 평소에 알기 힘든 아파트 관련 회계 정보를 손쉽게 제공함으로써 투명한 아파트 관리 시스템 구축

## 예상 기능 영역

- 커뮤니티 게시판
- 공지사항 게시판
- 시설 예약 시스템
- 민원 접수 및 처리 게시판
- 이웃 간 소통 기능
- 전자 고지서
- 전자 투표/설문
- 폐기물/폐가전 수거
- 아파트 내 중고거래
- 공동구매 제휴 기능
- 주차 현황
- 아파트 회계 정보 간소화
- 수리/보수 전문가 매칭
