# 공지사항 페이지 구현 완료

_작성일: 2025-11-04_

_작성자: Developer Agent_

## 📋 작업 개요

공지사항 목록 및 상세 페이지를 구현하여 관리사무소의 공지사항을 입주민들이 확인할 수 있도록 했습니다.

## ✅ 구현 내용

### 1. 공지사항 목록 페이지 (`/notices`)

**파일**: `src/app/notices/page.tsx`

**주요 기능**:
- 공지사항 목록 조회 (tRPC API 연동)
- 카테고리별 필터링 (전체, 일반, 시설관리, 행사, 안전)
- 긴급 공지 시각적 강조 (빨간색 좌측 테두리, AlertCircle 아이콘)
- 조회수 표시
- 반응형 디자인 (모바일 우선)

**UI 특징**:
- 긴급 공지: 빨간색 Badge + 좌측 빨간 테두리
- 일반 공지: 회색 Badge
- 카테고리 필터: Badge 형태의 탭
- 날짜 형식: YYYY.MM.DD

### 2. 공지사항 상세 페이지 (`/notices/[id]`)

**파일**: `src/app/notices/[id]/page.tsx`

**주요 기능**:
- 공지사항 상세 내용 조회
- 조회수 자동 증가 (API에서 처리)
- 뒤로가기 버튼
- 긴급 공지 시각적 강조

**UI 특징**:
- 긴급 공지: 제목 빨간색, 좌측 빨간 테두리
- 작성자: "관리사무소" 고정 표시
- 날짜 형식: YYYY년 M월 D일
- 조회수 표시

### 반응형
- 모바일 우선 설계
- 최대 너비: max-w-2xl (상세), max-w-4xl (목록)

## 🔗 API 연동

### 사용된 tRPC 쿼리

1. **notices.getList**
   - 입력: apartmentId, category, page, limit
   - 출력: items[], total, hasMore

2. **notices.getById**
   - 입력: id
   - 출력: Notice 객체 (author, apartment 포함)

## 📱 사용자 경험

### 목록 페이지
1. 카테고리 선택으로 필터링
2. 긴급 공지 우선 표시 (시각적 강조)
3. 카드 클릭 시 상세 페이지 이동

### 상세 페이지
1. 뒤로가기 버튼으로 목록 복귀
2. 전체 내용 표시 (줄바꿈 유지)
3. 조회수 자동 증가
